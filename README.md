WeCare - Modern Hospital & Patient Portal Website

WeCare is a modern hospital management and patient portal website designed to provide a seamless healthcare experience. Patients can explore medical services, browse departments, learn about doctors, and book appointments through a clean and intuitive interface.

The application uses Firebase Authentication for secure user login and registration, while Firestore stores patient information and appointment data securely in the cloud.

Features
Secure user authentication with Firebase (Sign Up, Login, Logout, Session Management)
Appointment booking with department, doctor, date, time, and description selection
Personal patient dashboard to view, manage, reschedule, or cancel appointments
Responsive Departments page showcasing hospital specialties
Doctors directory with professional profiles and experience
Clean, modern, and fully responsive UI
Smooth page transitions and animations
Firebase Firestore integration for storing users and appointments
Tech Stack
React 19
Vite
Tailwind CSS v4
Firebase Authentication
Firebase Firestore
Lucide React
Motion (motion/react) for animations
